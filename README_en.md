# City Wings

A Flappy Bird-style casual game set against the backdrop of a city skyline. Fly your plane through the skyscrapers and challenge your piloting skills.

![Game Screenshot](demo/FlappyPlane.png)

## How to Play

- **Controls**: Press `Space` to make the plane fly up, release to let gravity pull it down
- **Objective**: Navigate through as many building gaps as possible for a higher score
- **Game Over**: Collision with buildings, ground, or flying off the top of the screen

## Features

- City skyline background
- Plane trail and collision particle effects
- Progressive difficulty
- Local high score tracking

## How to Run

- **Web version**: Open `index.html` in the project root with any modern browser. No installation or dependencies required.
- **APK version (Android)**: The repository includes an `android-app` project that can be built and packaged as an APK for Android devices.
